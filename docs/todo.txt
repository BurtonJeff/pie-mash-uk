1.	[DONE] Remove the activity tab from groups screen and add a new feature to groups called 'Meatup'. This is for organising a meet-up with fellow members of the group at a shop.  Any member of a group should be able to propose a meet-up, specifying the shop, the date and the time. Checks should be made to make sure the date and time fit within the opening hours of the shop concerned. Once set up members of a group should be able to register they are going or un-register if they have already accepted.
2.	[DONE] In groups an admin should be able delete the current chat.
3.	[DONE] In groups a group admin should be able to remove users from the group. They should also be able to make other users a group admin of the group or remove their group admin ability. The original creator the group cannot never have their admin status removed.
4.	[DONE] When creating a group it should be able to specify if 'confirmation' is needed by a group admin for the user to be added to the group. This 'requires confirmation' option should be able to be added or removed from existing groups by a group admin.
5.	[DONE] Once a user has logged into the app they should never have to log in again unless they have explicitly signed out of the app using the sign out feature on the profile screen
6.	[DONE] On the community screen I would like a separate tab called 'Social'. We can move the current Facebook group link from the 'Groups' tab to the 'Social' tab. The links on the 'Social' tab should be maintainable by an admin on the admin panel.
7.	[DONE] When checking into a shop it should be possible to integrate with social media such as Facebook and Instagram to create a post for the check in.
